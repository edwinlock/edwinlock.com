# How to make your own academic page.

1. Go to https://github.com/edwinlock/edwinlock.com and fork the repository.
2. Change the name of your new repository to <yourdomain.com>.
3. Change the description of the repository.
4. Clone the repository onto your local machine.
5. Delete the CNAME file.
6. Turn on github pages feature in settings.
7. Open _config.yml and make necessary changes.
8. Make sure the url has an https:// prefix!
9. 
10. Push all the changes back to the repository.
